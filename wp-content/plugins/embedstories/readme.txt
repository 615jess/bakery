==EmbedStories==

Save and Embed Instagram Stories on Any Website.

[LEARN MORE](https://embedsocial.com/products/embedstories/) | [FREE TRIAL](https://embedsocial.com/admin/continue_plugin_purchase/stories/triallist)

This is a unique tool that can fetch and embed your Instagram stories on your WordPress website with a simple shortcode. 

Once you embed the code it will automatically update every time you post a new story on your Instagram account.

To start using EmbedStories, [click here to create an account](https://embedsocial.com/admin/continue_plugin_purchase/stories/triallist).

**Features**

* Fetch the latest story published on your instagram account
* Every-time you add a photo or video to the story, the web widget will reflect the changes, in just a few minutes.
* Automatically archive all stories
* Embed entire story on any website with a simple embeddable code
* Customize the look of the story slider
* Create story gallery from multiple Instagram stories

**Settings**

* show the latest Instagram story
* show story from a specific date
* order by date
* autoplay Interval
* Customize title
* Customize navigation

Check out the Live [DEMO](https://embedsocial.com/products/embedstories/#stories-demo) and see how an Instagram story displays on a website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/embestories` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. To use this plugin you need to have an account on [EmbedSocial.com](https://embedsocial.com)
